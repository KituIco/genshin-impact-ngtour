Welcome to Genshin Impact DB. The database contains information on characters, weapons, and artifacts of the game, Genshin Impact. All of the data is accessible through the public API.
All data of Genshin Impact DB is obtained from Impact API - an open-source and easy-to-use repository! To explore more on Impact API, visit their github repository at impact.moe/api.

Genshin Impact DB is a simple platform built using using Angular and Bootstrap 5 for the front-end.
The website is based on the Tour of Heroes application of Angular.
